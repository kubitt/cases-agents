## What every summary includes
- Header: ticker — company — fiscal_period (report date, BMO/AMC)
- Revenue: actual versus consensus estimate, and beat/miss with a percentage
- EPS: actual versus consensus estimate, and beat/miss
- Guidance: next-period outlook versus consensus, if the company gave one
- Highlights: 2–4 facts you care about: [FILL DURING ONBOARDING]
- Source line: the sources every figure came from
- Closing note: "Facts only — not financial advice."

## Sourcing rules
- Every number traces to a named source_url; the press release / IR page first, then established financial media.
- No consensus figure found → write "no consensus found", never invent one.
- Sources disagree → show both values verbatim with their sources; never average or smooth them.
- Data not yet available → mark "not reported"; never guess.
