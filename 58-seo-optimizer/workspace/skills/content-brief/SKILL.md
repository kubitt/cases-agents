# Content brief

> A skill = one repeatable job, written as a recipe. You can edit steps, limits, and formats in plain words.

Goal: give a writer everything needed to create a page that can rank — intent, structure, subtopics, and targets — in one Google Doc.

## Algorithm
1. Confirm the target keyword and its cluster from the "Keyword Map" tab of "SEO Tracker" (add it via skill keyword-research if missing).
2. Read the intent: run the search, read the top 5 live results, and describe what the searcher actually wants (a guide, a comparison, a tool, a definition).
3. Analyze the top results: for each, note format, approximate word count, and the H2/H3 topics they cover. List the subtopics and questions ALL of them cover (must-haves) and gaps none cover (your edge).
4. Recommend a heading structure: a working title, then H2/H3 outline that covers the must-haves plus the gap, in a logical order.
5. Set targets: word-count range (aim for the median of the top results, not the maximum), the target keyword and 3–6 supporting terms to use naturally, and 3–5 internal links from SITE_PROFILE.md priority pages.
6. Draft title (50–60 chars, keyword included) and meta description (140–160 chars) options per SEO_STANDARDS.md.
7. Save a Google Doc "SEO Brief — <keyword>" and note the link in the keyword's row (notes).

## Output
A Google Doc plus a short chat summary. Example (abbreviated):
- SEO Brief — how to reduce no-shows
- Intent: informational; searcher wants practical tactics, not software pitches.
- Working title: "How to Reduce No-Shows: 9 Tactics That Work" (44 chars)
- Outline: H2 Why no-shows happen; H2 Reminders that work; H2 Deposits and policies; H2 When to use a CRM.
- Targets: 1,400–1,700 words; terms: appointment reminders, SMS confirmations, cancellation policy; internal links: /dental-crm, /pricing.

## Edge cases
- Thin or off-topic top results → tell the user the keyword may be low-value or misread; propose a better target from the cluster.
- Mixed intent on page one (some guides, some product pages) → note it and recommend the format that fits the site's page type.
- Your own page already ranks in the top 5 → recommend refreshing it instead of writing new (avoid competing with yourself).
- Sensitive topic (health, finance, legal) → require expert review and cited sources in the brief; do not imply the agent's text is authoritative.
