# Интеграции: GEO-оптимизатор

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Docs | Обязательно | Отчёты аудита, переписанные страницы «было/стало», протоколы проверки видимости | Чтение и запись |
| Google Search Console | Рекомендуется | Реальные запросы и показы страниц: приоритизация, отслеживание вопросных запросов | Чтение |
| Google Sheets | Рекомендуется | Трекер видимости: 10 запросов x месяцы, оценки аудита, статусы внедрения правок | Чтение и запись |
| Telegram (канал) | Опционально | Ежемесячный отчёт видимости, готовность аудитов | Канал доставки |

Встроенные веб-поиск и чтение веб-страниц — основной инструмент (аудит страниц, проверка выдачи, анализ конкурентов), OAuth не требуют.

Порядок подключения при онбординге: Google Docs → Search Console → Sheets → Telegram.

Без Search Console: приоритизация страниц — по здравому смыслу и веб-поиску (какие страницы в выдаче), без данных о показах. Без Sheets: динамика видимости хранится текстом в Google Docs. Без Telegram: отчёты в чат ASCN.

Чего сознательно нет: API ChatGPT/Perplexity/Google AI Overviews для трекинга цитирований (таких коннекторов в ASCN нет, браузерной автоматизации нет) — заменено проверкой через веб-поиск + протоколом ручной проверки раз в месяц. Доступа к CMS нет — внедрение правок на стороне пользователя; продвинутый вариант: кастомный MCP к API вашей CMS.
