# Готовые агенты ASCN — премиум-линейка для маркетплейса

18 готовых к деплою агентов с высоким ROI (окупаемость подписки $29 — 15–100x). Идеи отобраны по измеренным данным библиотеки lib.cybos.ai (proven-скиллы и recipes) и адаптированы под возможности ASCN: 13 OAuth-интеграций, триггеры Gmail/Calendar/Drive, веб-поиск, без браузерной автоматизации.

## Структура каждой папки

```
NN-slug/
├── README.md          # витрина: ценность, ROI, что делает, ограничения
├── DEPLOY_PROMPT.md   # промпт для Claude Code — перенос в ASCN за один прогон
├── INTEGRATIONS.md    # какие интеграции, зачем, деградация без них
├── AUTOMATIONS.md     # готовые тексты cron-задач и триггеров
└── workspace/         # копируется в /data/workspace/ агента как есть
    ├── IDENTITY.md    # роль, функции, границы, форматы
    ├── SOUL.md        # характер + жёсткие правила безопасности
    ├── ONBOARDING.md  # пошаговый онбординг пользователя (state machine)
    ├── skills/        # 2 скилла с алгоритмами и краевыми случаями
    └── knowledge/     # 2 шаблона/плейбука (заполняются при онбординге)
```

## Как деплоить (5 минут на агента)

1. Создайте нового агента в ASCN (пустой воркспейс).
2. Откройте Claude Code в папке агента, вставьте содержимое его `DEPLOY_PROMPT.md`.
3. Claude Code перенесёт файлы (напрямую или через MCP воркспейса) и запустит первый диалог.
4. Дальше агент сам ведёт онбординг пользователя: OAuth-кнопки, персонализация, создание cron-задач, тестовый прогон на реальных данных, фиксация `onboarding_complete=true`.
