# Интеграции: Отчёт для руководства + HTML-дашборд

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Sheets | Обязательно | Источник большинства метрик + служебная таблица «KPI-история» | Чтение и запись |
| Google Drive | Обязательно | Хранение dashboard.html и месячных копий отчётов | Чтение и запись |
| Google Search Console | Опционально | Поисковый трафик: клики, показы, позиции | Чтение |
| GitHub | Опционально | Велосити разработки: merged PR, релизы, issues | Чтение |
| Slack или Telegram (канал) | Рекомендуется | Доставка отчётов по расписанию | Канал доставки |

Порядок подключения при онбординге: Sheets → Drive → (GSC) → (GitHub) → канал.

Без GSC/GitHub: соответствующие кластеры просто не появляются в отчёте. Без Slack/Telegram: отчёты приходят в чат ASCN. Без Drive агент работать не может — дашборду нужно место хранения.

Чего сознательно нет: Stripe, Mixpanel, Amplitude, CRM, рекламных кабинетов — их нет среди OAuth-коннекторов платформы. Замена: регулярные выгрузки в Sheets (агент подскажет формат листа) или кастомный MCP-сервер как расширенная настройка.
