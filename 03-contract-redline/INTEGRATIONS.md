# Интеграции: Юрист первой линии (redline договоров)

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Drive | Обязательно | Триггер на папку «Договоры», хранение исходников и готовых пакетов | Чтение и запись |
| Google Docs | Обязательно | Redline-копии с правками, risk-memo, чек-листы NDA | Чтение и запись |
| Gmail | Рекомендуется | Черновики писем контрагенту с правками (отправка после утверждения) | Черновики, отправка |
| Google Sheets | Рекомендуется | Реестр договоров: сроки, автопролонгации, вердикты, ссылки | Чтение и запись |
| Telegram (канал) | Рекомендуется | Алерты «договор разобран», дедлайны пролонгаций | Канал доставки |

Порядок подключения при онбординге: Drive → Docs → Sheets → Gmail → Telegram.

Без Sheets: нет реестра — ежедневная проверка сроков и пролонгаций не работает, остаётся разбор входящих. Без Gmail: тексты писем контрагенту выдаются в чат, отправляете сами. Без Telegram: уведомления в чат ASCN.

Чего сознательно нет: e-signature (DocuSign/КонтурДиадок — нет в коннекторах), юридических баз практики по API. Замена: подписание и проверка практики — у юриста-человека; агент готовит черновик решения. Продвинутый вариант — кастомный MCP к вашему СЭД.
