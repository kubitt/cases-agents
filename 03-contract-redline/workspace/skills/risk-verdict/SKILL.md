# Risk-memo и вердикт

Цель: одностраничный документ, по которому руководитель принимает решение за 2 минуты.

## Правила вердикта (по LEGAL_PLAYBOOK.md)
- ПОДПИСАТЬ: ноль критичных, ≤2 некритичных отклонений, все блоки покрыты позициями плейбука.
- ТОРГОВАТЬСЯ: есть отклонения, и по каждому в плейбуке есть целевая и запасная позиция.
- ЭСКАЛИРОВАТЬ: хотя бы один пункт «нет позиции», критичное без запасного варианта, тип договора вне профиля, сумма ≥ порога эскалации.
Сомнение между двумя вердиктами → более осторожный. Вердикт не смягчается по просьбе — только через изменение плейбука пользователем.

## Шаблон memo (сохраняется в Docs «<Контрагент> — memo <дата>», краткая версия в канал)
```
ДОГОВОР: ООО Прайм — услуги — 1 800 000 ₽ — 12 мес
ВЕРДИКТ: ТОРГОВАТЬСЯ
Топ-риски:
1. п. 7.2 — ответственность не ограничена — КРИТИЧНО. Правка: лимит = стоимость договора.
2. п. 4.1 — оплата 90 дней (плейбук: ≤30) — отклонение. Правка: 30 дней, запасная: 45.
3. п. 11.3 — автопролонгация с уведомлением за 10 дней — отклонение. Правка: 30 дней + строка в реестре.
Торг-план: обязательно к изменению — 7.2; желательно — 4.1, 11.3; можно уступить — 9.4 (подсудность по месту ответчика).
Дедлайн: контрагент ждёт ответ до 12.07.
Письмо контрагенту: черновик готов в Gmail — отправлю после вашего утверждения.
Дисклеймер: это черновой разбор, а не юридическая консультация. Финальное решение — за вашим юристом.
```

## Повторный раунд (новая редакция от контрагента)
1. Сравни новую версию с нашей redline: что приняли / что отклонили / что изменили по-своему (проверь на скрытые изменения в блоках, которые не обсуждались!).
2. Memo-дельта ≤10 строк: принято N из M, оставшиеся риски, новый вердикт.

## Краевые случаи
- Пользователь решает подписать вопреки «эскалировать» → зафиксируй в реестре «подписано с известным риском: <список>», один раз предупреди, не спорь дважды.
- Все правки приняты контрагентом → вердикт «подписать», напомни внести end_date и notice_deadline в реестр.
- Нет данных о сумме договора → порог эскалации по сумме не применяй, отметь «сумма не указана» в memo.
