# Юрист первой линии (redline договоров) / First-Line Contract Redliner

Входящий договор → redline-правки по каждому рискованному пункту и вердикт «подписать / торговаться / эскалировать» по плейбуку вашей компании. NDA — разворот в тот же день.

## Ценность и ROI

Юрист — $100–300/час, ревью одного NDA на аутсорсе — $150–500. Агент закрывает первую линию: типовые NDA и договоры разбирает сам, юристу-человеку уходят только реальные риски. 2–3 договора в месяц = 15–50x подписки $29/мес. Бонус: контроль автопролонгаций — одна незамеченная пролонгация ненужного контракта стоит дороже года подписки.

## Что делает

1. Ловит новые договоры в Drive-папке «Договоры» (триггер) или файлом в чат (PDF/DOCX).
2. Извлекает условия по 10 блокам: предмет, оплата, срок и автопролонгация, ответственность, неустойки, IP, конфиденциальность, расторжение, гарантии, подсудность.
3. Сверяет каждый блок с плейбуком компании (workspace/knowledge/LEGAL_PLAYBOOK.md): ок / отклонение / критично.
4. Готовит redline: копия в Google Docs с готовыми формулировками правок и комментарием «почему» к каждой.
5. Пишет одностраничный risk-memo с вердиктом: подписать / торговаться (с торг-планом) / эскалировать юристу.
6. NDA — ускоренный сценарий по NDA_FASTTRACK.md: чек-лист из 8 пунктов, правки, черновик ответа контрагенту в тот же день.
7. Ведёт реестр договоров в Sheets и ежедневно в 09:15 проверяет сроки: окончания договоров и дедлайны уведомлений об отказе от автопролонгации.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN (IDENTITY, SOUL, ONBOARDING, скиллы, плейбук, NDA fast-track).
- `DEPLOY_PROMPT.md` — промпт для Claude Code, который переносит всё в ASCN.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка по интеграциям и готовые тексты триггера и cron-задачи.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — дальше он сам проведёт онбординг (интеграции, плейбук позиций, разбор первого реального договора).

## Требуемые интеграции

Google Drive (обязательно, триггер), Google Docs (обязательно), Gmail и Google Sheets (рекомендуются). Канал: Telegram (рекомендуется).

## Ограничения (сознательные)

- Агент НЕ юрист и не даёт юридических консультаций: результат — черновик решения с обязательным дисклеймером. Финальное слово — за юристом-человеком.
- Не редактирует оригинал файла — работает только с копией «— REDLINE» в Google Docs.
- Профиль по умолчанию — типовые коммерческие договоры; гос.закупки, M&A, международные споры, регулируемые лицензии — сразу эскалация без разбора.
- Без e-signature (DocuSign и аналогов нет в коннекторах ASCN) — подписание вне агента; агент лишь фиксирует статус в реестре.
