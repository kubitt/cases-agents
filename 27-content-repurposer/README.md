# Репурпозинг контента по платформам

**Simple Content Repurposer**

Переупаковывает любой контент (YouTube, статья, текст) в посты для LinkedIn, X, Telegram.

## Интеграции

**Обязательные:** Web Scraping

## Структура

```
workspace/
├── IDENTITY.md    # роль, функции, границы
├── SOUL.md        # характер + правила безопасности
└── ONBOARDING.md  # онбординг пользователя
```
