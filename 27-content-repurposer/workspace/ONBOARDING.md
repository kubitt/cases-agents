# Онбординг — Simple Content Repurposer

> Сценарий первичной настройки. Агент проходит его по шагам при первом контакте. После настройки файл обычно не правят.

При первом запуске агент собирает данные пользователя.

## Данные для заполнения

# О пользователе
Имя: {заполни в чате}
Обращение: {на "ты" / на "вы"}
Часовой пояс: {например, UTC+3}
Язык: русский

## Бизнес
Компания: {название, если есть}
Сфера: {чем занимаешься}
Роль: {основатель / маркетолог / эксперт / автор}

## Контент
Площадки: {LinkedIn / X / Telegram — какие активно ведёшь}
Тематика: {о чём пишешь в одной-двух фразах}
Цель контента: {личный бренд / лидген / экспертиза / комьюнити}

## Стиль
Образцы стиля: {ключевые наблюдения — длина постов, тон, любимые обороты, структура}
Любимые форматы: {треды / лонгриды / короткие заметки / разборы}
Чего избегать: {слова, темы, штампы которые тебе не нравятся}

## Предпочтения
Длина LinkedIn: {короткие до 500 знаков / средние / лонгриды}
X — треды или одиночные посты: {треды / одиночные / по ситуации}
Telegram — личный или экспертный тон: {личный / экспертный / микс}
