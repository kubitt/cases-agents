# Интеграции: Бейджи с выставки → пайплайн

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Drive | Обязательно | Папка «Выставки»: триггер на новые CSV со сканами | Чтение |
| Google Sheets | Обязательно | Пайплайн мероприятия: обогащение, скоринг, статусы, конверсия | Чтение и запись |
| Gmail | Обязательно | Черновики follow-up, отправка после утверждения, отслеживание ответов | Чтение, черновики, отправка |
| Telegram (канал) | Рекомендуется | Батчи на утверждение в день выставки, алерты об ответах | Канал доставки |

Порядок при онбординге: Sheets → Gmail → Drive → Telegram.

Без Drive: файлы кидаются прямо в чат (триггер не работает, запуск командой). Без Telegram: всё в чат ASCN.

Чего сознательно нет: CRM-коннекторов (экспорт из Sheets или кастомный MCP), сендеров (отправка через Gmail: для пост-ивентных объёмов 50–200 писем это нормально, с разносом по дням).
