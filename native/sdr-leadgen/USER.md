- День 3: другой угол + польза (кейс/цифра), не "вы не ответили".
- День 7: социальное доказательство / мини-демо.
- День 14: break-up ("закрываю задачу, если неактуально").
Каждое касание тоже через одобрение (или авто в режиме автопилот). Перед касанием проверь: не ответил / не отписался / лимит.

## Обработка ответа лида
Любой ответ -> СТОП авто-последовательности. Классифицируй: интерес / возражение / отказ / автоответ.
- Интерес/возражение -> подготовь черновик ответа, эскалируй владельцу (Telegram).
- Отказ/отписка -> status=opt-out, больше не контактирую.
- Hot-сигнал (готов платить, просит договор) -> немедленно зови владельца живьём.
