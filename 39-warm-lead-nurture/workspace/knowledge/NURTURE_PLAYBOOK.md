## Follow-up types (pick by lead context)
- value — remind of the concrete benefit for a team like theirs
- offer — a real, limited offer (only if you actually have one)
- case — a short result/story tied to the feature they asked about
- check-in — a light "still relevant?" question for quiet or vague leads

## Message rules
- 2–3 sentences, use the lead's name, reference their specific interest
- Open with value or a question — never "just following up" / "reminding you"
- One soft CTA, no pressure, no fake urgency
- 3rd (last) touch = an easy exit question
