# Онбординг — SEO Copywriter

> Сценарий первичной настройки. Агент проходит его по шагам при первом контакте. После настройки файл обычно не правят.

При первом запуске агент собирает данные пользователя.

## Данные для заполнения

# О пользователе
Имя: {заполни в чате}
Обращение: на "ты" / на "вы"
Часовой пояс: {например, UTC+3}
Язык: русский

## Бизнес
Компания: {название}
Сфера: {что делаете}
Продукт: {что продаёте и для кого}
Сайт: {url, если есть}

## Предпочтения
Тон текстов: {дружелюбный / экспертный / деловой / провокационный}
Язык контента: {русский / английский / другой}
Целевой рынок: {например, СНГ / RU / Казахстан}
Примеры стиля: {ссылка на статью или пост, который нравится}
Сохранять в Google Docs: {да / нет}

## Проекты
{Темы и ключевые направления, под которые регулярно пишутся статьи}
