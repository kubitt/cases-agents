# Инвестор-апдейт и борд-дек / Investor Update & Board Deck

Send-ready письмо инвесторам из реальных цифр в Sheets — с честными lowlights и конкретными asks; по запросу — борд-дек на 12–15 слайдов со speaker notes.

## Ценность и ROI

Фаундер тратит 4–8 часов на каждый апдейт — поэтому апдейты не выходят, а молчание фаундера инвесторы читают как плохие новости. Регулярный честный апдейт = доверие, интро и follow-on раунды. Агент готовит черновик за минуты из ваших же таблиц. При подписке $29/мес окупаемость 15–30x времени фаундера, эффект на отношения с инвесторами — больше.

## Что делает

1. 1-го числа в 10:00 — черновик месячного апдейта: TL;DR, таблица метрик (MRR/ARR, burn, cash, runway, пользователи) с MoM-дельтами, highlights, lowlights (обязательно), конкретные asks, follow-up по прошлым asks.
2. Цифры берёт только из ваших Sheets по карте knowledge/METRICS_SOURCES.md — каждая трассируется до ячейки.
3. Перекрёстная сверка: runway пересчитывается из cash и среднего burn; расхождения с заявленными цифрами показываются явно отдельным блоком.
4. Тон — из ваших прошлых апдейтов (архив ведётся в Sheets).
5. Письмо — черновиком в Gmail; отправка только после ваших правок и явной команды «отправляй».
6. Борд-дек по команде: 12–15 слайдов в Google Slides — заголовки-тезисы, буллеты, speaker notes и черновик контента каждого слайда.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN (IDENTITY, SOUL, ONBOARDING, скиллы, шаблоны).
- `DEPLOY_PROMPT.md` — промпт для Claude Code, который переносит всё в ASCN.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка по интеграциям и готовые тексты cron-задач.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — дальше он сам проведёт онбординг (интеграции, источники метрик, тон, тестовый черновик).

## Требуемые интеграции

Google Sheets (обязательно), Gmail (обязательно). Google Slides + Drive — для борд-дека (опционально).

## Ограничения (сознательные)

- Нет интеграций с банками/Stripe/QuickBooks — цифры живут в Sheets (выгрузки или ручное ведение раз в месяц, ~10 минут).
- Агент не приукрашивает: убрать lowlights или «округлить» метрики он откажется — это зашито в SOUL. Формулировки править можно, факты — нет.
- Борд-дек — черновик структуры и контента в Slides; финальный дизайн и расстановку акцентов делаете вы.
