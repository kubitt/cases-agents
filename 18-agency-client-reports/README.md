# Мульти-клиент отчёты для агентств / Agency Client Reports

Для агентств (SEO/SMM/перфоманс): месячные брендированные отчёты всем клиентам — за одну сессию вместо недели ручной сборки. Данные из Search Console и ваших таблиц, выводы человеческим языком, рассылка после утверждения.

## Ценность и ROI

Сервисы отчётности (AgencyAnalytics, Swydo) стоят $150–400/мес, и отчёты всё равно дособирают руками: 2–4 часа на клиента. Агентство с 10 клиентами тратит 20–40 часов/мес только на отчётность. Агент собирает батч за сессию. Окупаемость $29: 30–80x.

## Что делает

1. Ведёт реестр клиентов: домен GSC, папка Drive, контакт, особенности, статус отчёта.
2. По каждому клиенту собирает отчёт: динамика поискового трафика (GSC), выполненные работы (из вашей таблицы работ), план на следующий месяц, выводы «что это значит» без жаргона.
3. Оформляет в Google Docs по вашему брендированному шаблону — структура и тон настраиваются один раз.
4. Батч-режим: 1-го числа проходит всех активных клиентов, складывает отчёты по папкам Drive, даёт сводку на утверждение.
5. После «ок» — черновики писем каждому клиенту (персональные, не «рассылка»), отправка после утверждения.
6. Трекинг: кто получил, кто открыл вопросами вернулся — статусы в реестре.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN.
- `DEPLOY_PROMPT.md` — промпт для Claude Code.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка и текст батч-задачи.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — онбординг: реестр клиентов, шаблон, тестовый отчёт по одному клиенту.

## Требуемые интеграции

Google Search Console, Google Sheets, Google Docs, Google Drive, Gmail (все — обязательно для полного цикла). Канал: Telegram (рекомендуется).

## Ограничения (сознательные)

- Рекламные кабинеты (Яндекс.Директ, Google Ads, Meta) не подключаются нативно — метрики рекламы попадают в отчёт из ваших выгрузок в Sheets (колонки настраиваются) или через кастомный MCP.
- GSC должен быть подключён к аккаунту с доступом к доменам клиентов (обычная практика агентств).
- Соцсети — аналогично: цифры из ваших выгрузок, автоматического сбора нет.
