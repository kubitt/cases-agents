# Автоматизации: Мульти-клиент отчёты

## Cron-задачи

### 1. Месячный батч отчётов
- Cron: `0 9 1 * *` (1-е число, 09:00)
- Промпт задачи: «Открой Sheets "Реестр клиентов". Для каждого клиента со status=active собери месячный отчёт по скиллу client-report за прошедший календарный месяц. Складывай в Drive: папка клиента → "Отчёты/<YYYY-MM>". По завершении батча пришли в Telegram сводку по скиллу multi-client-orchestrator: клиент — ключевой итог месяца одной строкой — ссылка на отчёт — флаг (норма/просадка/рост). Спроси подтверждение на подготовку писем. После утверждения — черновики Gmail каждому клиенту (персональное вступление по итогам его месяца), отправка после финального "ок" пользователя. Обнови статусы в реестре.»
- Канал: Telegram.

### 2. Напоминание о выгрузках (за 2 дня до батча)
- Cron: `0 12 28-30 * *` (28–30 числа, 12:00 — агент сам проверяет, что до конца месяца ≤2 дней)
- Промпт задачи: «Проверь по реестру, у каких активных клиентов в Sheets не обновлены листы ads_data/social_data за текущий месяц (если для клиента настроены эти блоки). Пришли пользователю список одним сообщением: "для полных отчётов не хватает выгрузок: …". Если всё на месте — ничего не отправляй.»
- Канал: Telegram.

## Триггеры
Нет обязательных. Опционально (создаётся по желанию на онбординге): Gmail — ответ клиента на письмо с отчётом → классифицировать (вопрос/ок/недовольство), при вопросе или недовольстве — алерт с черновиком ответа.

Тихие часы 23:00–08:00 соблюдаются.
