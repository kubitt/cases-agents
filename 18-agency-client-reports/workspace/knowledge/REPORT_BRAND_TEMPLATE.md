# Брендированный шаблон отчёта
[Структура и тон — ЗАПОЛНЯЮТСЯ ПРИ ОНБОРДИНГЕ по примеру отчёта агентства; ниже дефолтный каркас]

## Титул
<Логотип/имя агентства> — Отчёт для <Клиент>, <месяц год>. Одной фразой: главный итог месяца.

## 1. Цифры месяца (полстраницы)
Таблица: метрика | месяц | прошлый | Δ% | комментарий одной фразой. Только метрики из blocks клиента. Стрелки ▲▼ у значимых изменений (>10%).

## 2. Поиск (GSC)
Динамика кликов/показов/позиции; топ-5 растущих запросов; что просело и почему; новые страницы в топ-10.

## 3. Что сделано
Группы работ с «зачем»: не «размещено 12 ссылок», а «усилили 3 денежные страницы — вот их рост».

## 4. [Реклама] / [Соцсети] — если включены
Из выгрузок: расход, лиды, CPL, динамика. Данных нет → «данные за период не предоставлены».

## 5. Выводы и план
3–5 выводов человеческим языком → план следующего месяца (3–5 работ с ожидаемым эффектом).

## Тон
[из примера пользователя: обращение, степень уверенности, любимые обороты]
Запрещено: канцелярит, необъяснённые термины, оправдательный тон, обещания конкретных позиций.
