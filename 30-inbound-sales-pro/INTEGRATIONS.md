# Интеграции — Inbound Sales Pro

## Обязательные

### Telegram
**Зачем:** основной канал входящих продаж — клиенты пишут сюда, агент отвечает.
**Два режима:**
- **Бот** — создаётся через @BotFather, токен вставляется в Каналы → Telegram.
- **Личный аккаунт** — агент подключается к Telegram-аккаунту владельца и отвечает от его имени.
**Подключение:** Каналы → Telegram → выбрать режим → следовать инструкциям.

## Рекомендуемые

### Gmail
**Зачем:** ловить заявки из почты, отправлять follow-up.
**Подключение:** Мозг → Подключения → Gmail → + Добавить → OAuth.

### Google Sheets
**Зачем:** простая CRM-таблица с лидами: имя, контакт, стадия воронки, скоринг.
**Подключение:** Мозг → Подключения → Google Sheets → + Добавить → OAuth.

### Notion
**Зачем:** полноценная CRM / база сделок. Альтернатива Google Sheets.
**Подключение:** Мозг → Подключения → Notion → + Добавить → OAuth.

### Google Calendar
**Зачем:** бронировать демо и звонки с клиентами.
**Подключение:** Мозг → Подключения → Google Calendar → + Добавить → OAuth.

## Опциональные

- **Slack** — уведомлять команду о горячих лидах.
- **Google Drive / Docs** — открывать прайсы, договоры.
- **Кастомные MCP-серверы** — любой внешний сервис через URL.
