# Структура месячного пакета

> Ваш справочный файл. Агент читает его перед работой и дополняет по ходу. Правьте свободно.

## Одностраничник для основателя (Google Docs)
1. Заголовок: «Финансы <месяц>: <одна фраза-вывод>» (пример: «Март: рост выручки +18%, но кассовый разрыв в мае при текущем графике оплат»).
2. Пять цифр крупно: Выручка (Δ к прошлому месяцу), Расходы (Δ), Чистый поток, Остаток на счетах, Runway (base).
3. Три отклонения месяца: что / насколько / вероятная причина / что делаем.
4. Дебиторка: сумма, топ-3 должника, просрочка.
5. Решения, которые нужны от основателя: 1–3 пункта с дедлайном.

## Sheets «Финпакет YYYY-MM»
- Лист P&L: строки = категории из CHART_OF_ACCOUNTS.md, колонки = факт месяца, прошлый месяц, среднее 3 мес, Δ%, комментарий.
- Лист CashFlow: поступления/списания по неделям месяца, остаток на конец каждой недели.
- Лист «Разобрать»: нераспознанные операции (дата, сумма, назначение) — пользователь проставляет категории, агент учится (дополняет правила в CHART_OF_ACCOUNTS.md).

## Правила подачи
Проценты — с одним знаком после запятой. Отрицательные — в скобках. Никаких «примерно» без пометки источника неточности. Расхождения сверки — всегда отдельным блоком, даже если мелкие.
