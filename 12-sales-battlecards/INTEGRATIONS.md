# Интеграции: Battlecards для отдела продаж

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Sheets | Обязательно | Реестр конкурентов + снимки для еженедельного диффа | Чтение и запись |
| Google Docs | Обязательно | Карточки: одна карточка = один док, расшаривается отделу продаж | Чтение и запись |
| Slack или Telegram (канал) | Рекомендуется | Пятничный дайджест изменений, запросы на утверждение | Канал доставки |
| Google Drive | Опционально | Папка «Battlecards» для порядка в доках | Чтение и запись |

Веб-поиск и чтение страниц — встроенные возможности платформы, отдельного подключения не требуют.

Порядок подключения при онбординге: Sheets → Docs → (Drive) → канал.

Без Slack/Telegram: дайджесты приходят в чат ASCN. Без Drive: доки создаются в корне «Мой диск».

Чего сознательно нет: коннекторов Crayon/Klue/Semrush и API соцсетей — мониторинг построен на веб-поиске по открытым страницам. Автотрекинг упоминаний и рекламы конкурентов при необходимости добавляется кастомным MCP-сервером.
