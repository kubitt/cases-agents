# Battlecards для отдела продаж / Sales Battlecards (опубликовано в marketplace)

> 📢 Опубликован в ASCN Marketplace: [RU](https://ascn.ai/ru/templates/ai-sales-battlecards-competitive-intelligence) · [EN](https://ascn.ai/en/templates/ai-sales-battlecards-competitive-intelligence)

Боевые карточки по каждому конкуренту из открытых источников + еженедельный мониторинг изменений — продавцы всегда знают, что отвечать на «а чем вы лучше X?».

## Ценность и ROI

Разовый конкурентный анализ у консультантов — $1000–3000, платформы типа Crayon/Klue — от $500/мес. Поддержание карточек вручную съедает 2–4 часа PMM в неделю, поэтому в большинстве команд они устаревают за квартал. Агент собирает и обновляет карточки сам. При подписке $29/мес окупаемость 30–100x: одна сделка, спасённая правильным ответом на возражение про конкурента, покрывает годы подписки.

## Что делает

1. По каждому конкуренту собирает боевую карточку из открытых источников: сильные/слабые стороны, цены, позиционирование, свежие новости.
2. Пишет скрипты «что говорить, когда клиент называет их» (3 сценария) и landmines-вопросы — вопросы клиенту, подсвечивающие слабости конкурента.
3. Хранит карточки в Google Docs (одна карточка = один док, удобно расшарить отделу продаж); реестр конкурентов и снимки — в Google Sheets.
4. Каждую пятницу в 15:00 сравнивает конкурентов с прошлым снимком: что изменилось (цены, продукты, мессаджинг) → дайджест «какие карточки обновить».
5. Правки в карточки вносит только после вашего утверждения; каждая карточка ведёт changelog.
6. По запросу: экспресс-карточка нового конкурента, сравнение «мы vs X» на одну страницу под конкретную сделку.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN (IDENTITY, SOUL, ONBOARDING, скиллы, шаблоны).
- `DEPLOY_PROMPT.md` — промпт для Claude Code, который переносит всё в ASCN.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка по интеграциям и готовый текст cron-задачи.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — дальше он сам проведёт онбординг (интеграции, список конкурентов, ваше позиционирование, первая карточка).

## Требуемые интеграции

Google Sheets (обязательно), Google Docs (обязательно). Веб-поиск встроен. Канал: Slack или Telegram (рекомендуется).

## Ограничения (сознательные)

- Только открытые источники: сайты, прайсы, отзывы, вакансии, новости. Браузерной автоматизации нет — агент не логинится в продукты конкурентов и не скрейпит закрытые площадки.
- Закрытые цены («по запросу») помечаются оценкой из отзывов/обзоров с источником, а не выдуманной цифрой.
- Нет автотрекинга соцсетей и рекламных кабинетов конкурентов — мониторинг веб-поиском по расписанию; при желании добавляется кастомным MCP.
