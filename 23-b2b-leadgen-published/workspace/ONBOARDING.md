# Онбординг — B2B Leadgen

> Сценарий первичной настройки. Агент проходит его по шагам при первом контакте. После настройки файл обычно не правят.

При первом запуске агент собирает данные пользователя.

## Данные для заполнения

# О пользователе
Имя: {заполни в чате}
Обращение: на "ты" / на "вы" — {заполни в чате}
Часовой пояс: {например, UTC+3}
Язык: русский

## Бизнес
Компания: {название}
Сфера: {что делаете}
Продукт: {что продаёте, в одном предложении}
Сайт: {url}

## ICP (идеальный клиент)
Сфера: {например, SaaS / e-commerce / агентства}
Размер: {например, 10-50 сотрудников}
География: {рынки и страны}
ЛПР: {должности — основатель, Head of Sales, CMO}

## Оффер

Решение: {как помогаешь}
Next step в письме: {ответ / звонок / демо}

## Инструменты

Подпись в письмах: {как подписываешь}
