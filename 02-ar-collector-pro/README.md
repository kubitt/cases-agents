# AR-коллектор PRO / AR Collector Pro

Возвращает дебиторку без вашего участия: ведёт напоминания по нарастающим стадиям, разбирает ответы должников и помнит каждое «оплатим в пятницу» — и возвращается к нему.

## Ценность и ROI

Пара часов в неделю на «выбивание» оплат — это $200–400/мес времени владельца или бухгалтера. Возврат даже одного зависшего счёта на $500–5000 окупает годовую подписку $29/мес. Просрочка старше 90 дней собирается вдвое хуже — системные касания с первого дня возвращают деньги, пока они собираемы. Окупаемость 20–170x.

## Что делает

1. Ведёт реестр счетов в Google Sheets: суммы, сроки, стадии, обещания, история касаний (схема — workspace/knowledge/REGISTER_SCHEMA.md).
2. Ежедневно в 11:00 проходит по реестру: кому пора касание — готовит черновик письма нужной стадии (S0 «до срока» → S4 «финальное»). Тон ужесточается по стадиям, но без угроз.
3. Разбирает ответы должников (триггер Gmail): оплатил / обещал / спорит / другое — статус и лог в реестре.
4. Трекает обещания: «оплатим в пятницу» → пауза до пятницы, на следующий день — проверка и вежливый возврат к договорённости, если денег нет.
5. Спорные счета не дожимает рассылкой: собирает суть спора и эскалирует вам с вариантами решения.
6. Пятничный отчёт: дебиторка по корзинам старения (0–30/31–60/61–90/90+), собрано за неделю, обещания на следующую, кандидаты на эскалацию.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN (IDENTITY, SOUL, ONBOARDING, скиллы, плейбук и схема реестра).
- `DEPLOY_PROMPT.md` — промпт для Claude Code, который переносит всё в ASCN.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка по интеграциям и готовые тексты cron-задач и триггера.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — дальше он сам проведёт онбординг (интеграции, реестр, тон касаний, тестовый прогон по реальным просрочкам).

## Требуемые интеграции

Gmail (обязательно), Google Sheets (обязательно). Канал: Telegram (рекомендуется).

## Ограничения (сознательные)

- Без интеграции с бухгалтерскими системами (QuickBooks/1С нет в коннекторах ASCN) — источник правды: реестр в Sheets, пополняется CSV-выгрузкой или вручную. Продвинутый вариант — кастомный MCP к вашей бухгалтерии.
- Не звонит должникам (платформа не умеет звонки) — только email-касания и отчёты вам.
- Не юридическое взыскание: без претензий, угроз судом и пеней «на глаз». Стадия S4 — передача вопроса вам для решения об эскалации.
- Факт оплаты агент сам не видит (нет доступа к банку): помечает «оплачено» только после вашего подтверждения или строки оплаты в реестре.
