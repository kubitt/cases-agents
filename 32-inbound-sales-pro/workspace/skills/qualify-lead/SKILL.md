# Скилл: Квалификация лида

За 2-3 вопроса определить, подходит ли клиент и насколько «горячий».

## Критерии (каждый 0-25, итого 0-100)

### Fit (соответствие ЦА) — 0-25
- 25: точное попадание в ЦА из knowledge/service.md
- 15: частичное совпадение
- 5: слабое совпадение
- 0: явно не ЦА

### Need (потребность) — 0-25
- 25: чётко описал проблему, которую мы решаем
- 15: проблема есть, но нечёткая
- 5: «просто интересуюсь»
- 0: нет потребности

### Budget (бюджет) — 0-25
- 25: бюджет назван, совпадает с ценами
- 15: контекст говорит о готовности
- 5: ищет бесплатное/дешёвое
- 0: явно не готов платить

### Urgency (срочность) — 0-25
- 25: «нужно вчера», конкретные дедлайны
- 15: «в ближайшее время»
- 5: «присматриваюсь»
- 0: нет срочности

## Категории

- **80-100 (HOT)** — веди к закрытию, приоритет
- **50-79 (WARM)** — презентуй, прогревай
- **25-49 (COLD)** — дай информацию, предложи вернуться
- **0-24 (NOT FIT)** — вежливо откажи

## Как квалифицировать

Вплетай в разговор:
1. «Какую задачу хотите решить?» → Need
2. «Для кого — для себя, для компании?» → Fit
3. «Когда планируете начинать?» → Urgency
4. Budget определяй по контексту или: «Есть ориентир по бюджету?»

Скоринг делай молча — клиенту не говори его «балл».
