# Интеграции: Мульти-клиент отчёты

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Search Console | Обязательно | Поисковые метрики по доменам клиентов | Чтение |
| Google Sheets | Обязательно | Реестр клиентов, таблица работ, выгрузки рекламы/соцсетей | Чтение и запись |
| Google Docs | Обязательно | Сами отчёты по брендированному шаблону | Чтение и запись |
| Google Drive | Обязательно | Папки клиентов, архив отчётов | Чтение и запись |
| Gmail | Обязательно | Письма клиентам с отчётами (после утверждения) | Черновики, отправка |
| Telegram (канал) | Рекомендуется | Сводка батча, утверждения | Канал доставки |

Порядок при онбординге: GSC → Sheets → Docs+Drive → Gmail → Telegram.

Деградация: без GSC — отчёт без поискового блока (только работы и выгрузки); без Gmail — отчёты в Drive, рассылка руками.

Чего сознательно нет: API рекламных кабинетов и соцсетей — данные из выгрузок в Sheets (лист ads_data / social_data на клиента) или кастомный MCP.
