# Майнер болей аудитории / Pain Miner

Находит, какими словами реальные люди описывают боли, которые закрывает ваш продукт, — из Hacker News, Reddit, форумов и Stack Exchange — и превращает это в ранжированные инсайты с дословными цитатами и готовыми углами для лендингов и контента.

## Ценность и ROI

Кастдев-интервью и агентство под позиционирование стоят $1000–5000 за проект и недели времени. Агент за одну сессию отдаёт кластеры болей с сотнями реальных цитат, языком аудитории для SEO и готовыми углами лендинга/контента. Окупаемость $29: десятки раз. Бонус: заголовки словами клиентов конвертируют лучше маркетинговой прозы.

## Что делает

1. Собирает обсуждения по вашим темам с открытых площадок (Hacker News и Stack Exchange — без ключей; Reddit и форумы — опционально), фильтруя по маркерам боли.
2. Извлекает из каждого обсуждения суть боли, дословную цитату, что человек хочет сделать, упомянутые инструменты и силу фрустрации.
3. Группирует боли в кластеры с частотностью, трендом и разбивкой по источникам.
4. По каждому кластеру даёт язык аудитории (для SEO и заголовков), упомянутых конкурентов, угол для лендинга и тему для контента.
5. Ранжирует кластеры по приоритету (размер × свежесть × вовлечённость) и отдаёт отчёт (insights.md/csv/json), а по подтверждению — в Google Таблицу и Telegram-дайджест.
6. Раз в неделю обновляет картину болей (опционально).

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN (IDENTITY, SOUL, ONBOARDING, skills/, knowledge/).
- `DEPLOY_PROMPT.md` — промпт для Claude Code.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка и тексты cron-задач.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — онбординг сам проведёт (домен, темы, тестовые кластеры по Hacker News сегодня).

## Требуемые интеграции

Никаких обязательных: Hacker News и Stack Exchange работают без ключей, веб-поиск и чтение страниц встроены. Опционально — Google Sheets/Docs (выгрузка) и Telegram (дайджест). Reddit — по желанию (ключи в Секретах).

## Ограничения (сознательные)

- Только дословные цитаты: если точную цитату не удалось подтвердить в тексте — элемент отбрасывается, не «додумывается».
- Ники и профили не хранятся — только обезличенный хеш; удалённый на источнике контент удаляется и у агента.
- Внешние записи (Таблицы/Telegram) — только по явному подтверждению.
- Reddit на бесплатном тарифе — некоммерческий ресёрч; X/Twitter по умолчанию выключен (платное чтение).
