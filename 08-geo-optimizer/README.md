# GEO-оптимизатор / GEO Optimizer

Делает ваш сайт цитируемым источником для ChatGPT, Perplexity и AI Overviews: аудит цитируемости, переписанные страницы, ежемесячная проверка видимости.

## Ценность и ROI

GEO-агентства берут $500–2000/мес за то же самое: аудит, правки контента, отчёты видимости. Ниша свежая, конкуренции мало, а трафик из AI-ответов уже отбирает долю у классического поиска. При подписке $29/мес окупаемость 17–70x: один клиент, пришедший «мне вас ChatGPT посоветовал», покрывает годы подписки.

## Что делает

1. Аудит сайта на цитируемость AI-поисковиками по 5 измерениям: доступность для AI-краулеров (llms.txt, robots.txt), структура ответов, факты и определённость, разметка (schema/FAQ), авторитетность. Оценки 0–2 по каждому пункту чек-листа + приоритизированный план правок.
2. Переписывает страницы под цитируемость: прямой ответ в первом абзаце, факты с числами, structured quotables (самодостаточные цитируемые абзацы), FAQ-блок, черновик schema-разметки. Выход — Google Docs «было/стало».
3. Ведёт файл фактов бренда (BRAND_FACTS.md): проверенные цифры и формулировки, которые вшиваются в каждую страницу.
4. Раз в месяц проверяет видимость через веб-поиск по 10 ключевым запросам ниши: кого цитируют источники AI-ответов, есть ли вы среди них, динамика к прошлому месяцу.
5. Citation-gap анализ: какие страницы конкурентов попадают в выдачу-источники и чего не хватает вашим.
6. По данным Google Search Console (если подключена) — какие страницы уже получают показы по вопросным запросам и их стоит оптимизировать первыми.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN (IDENTITY, SOUL, ONBOARDING, скиллы geo-audit и geo-content-rewrite, справочники GEO_CHECKLIST и BRAND_FACTS).
- `DEPLOY_PROMPT.md` — промпт для Claude Code, который переносит всё в ASCN.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка по интеграциям и готовый текст cron-задачи.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — дальше он сам проведёт онбординг (сайт, ниша, запросы, факты бренда, тестовый аудит одной страницы).

## Требуемые интеграции

Google Docs (обязательно), Google Search Console (рекомендуется), Google Sheets (рекомендуется). Плюс встроенные веб-поиск и чтение страниц.

## Ограничения (сознательные)

- Нет автоматического трекинга внутри ChatGPT/Perplexity/AI Overviews — их API в ASCN отсутствуют, браузерной автоматизации нет. Видимость проверяется косвенно: веб-поиском по ключевым запросам + вашей периодической ручной проверкой в самих чат-ботах (агент даёт готовый протокол проверки).
- Правки на сайт агент не внедряет — нет доступа к CMS. Выход: готовые тексты, разметка и инструкции; публикует пользователь (или его разработчик). Вариант для продвинутых: кастомный MCP к API вашей CMS.
- Обещать «гарантированное попадание в ответы ChatGPT» нельзя — алгоритмы закрыты; агент повышает вероятность цитирования по измеримым факторам.
