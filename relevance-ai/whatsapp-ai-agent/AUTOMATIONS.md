# Задачи и триггеры

## Триггер запуска (оригинальный)
По расписанию: каждые 10 минут
`Get new messages from all chats (every 10 mins)`

## Шаги выполнения
1. Get All Chats — получить список всех активных чатов
2. Get All Messages From Chat — проверить непрочитанные сообщения
3. Сформировать ответ (LLM + системный промпт)
4. Send Message In A Chat — отправить ответ обратно

## В формате ASCN (когда появится WhatsApp)

Написать агенту в чате:
```
Настройся как WhatsApp-ассистент. Каждые 10 минут проверяй новые сообщения
во всех WhatsApp чатах и отвечай на непрочитанные. Используй дружелюбный тон,
помни историю переписки, сохраняй новые контакты в память.
```

## Улучшение для ASCN

Вместо polling (каждые 10 мин) — использовать webhook-триггер:
"При получении нового WhatsApp сообщения → немедленно ответить"
Это снизит нагрузку и ускорит ответ.
