## Описание

Персональный WhatsApp-ассистент который работает 24/7. Автоматически отвечает на входящие сообщения, помнит историю переписки, сохраняет контакты и помогает управлять чатами.

Запускается по расписанию (каждые 10 минут) — проверяет новые сообщения и отвечает на непрочитанные.

## Для кого

- Малый бизнес который ведёт клиентов через WhatsApp

## Пример использования

1. Клиент пишет в WhatsApp: "Сколько стоит доставка?"
2. Агент получает сообщение через Get All Messages
3. Формирует ответ на основе системного промпта + базы знаний
4. Отправляет ответ через Send Message In A Chat
