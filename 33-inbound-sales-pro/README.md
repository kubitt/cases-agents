**Sales Pro**

ИИ-агент входящих продаж: квалифицирует лидов, презентует оффер, обрабатывает возражения по фреймворку ARPA, доводит до сделки. Работает в Telegram (бот или личный аккаунт). Владелец перехватывает любой диалог командой /stop.

## Ценность и ROI

Менеджер по продажам тратит 60-70% времени на квалификацию и типовые вопросы. Агент берёт на себя первую линию 24/7: отвечает за секунды, квалифицирует по скорингу, передаёт менеджеру только горячих. При потоке 20+ лидов/день — экономия 4-6 часов менеджера ежедневно.

## Что делает

1. Принимает входящие сообщения в Telegram (бот или личный аккаунт).
2. Квалифицирует лида по 4 критериям (Fit, Need, Budget, Urgency) → скоринг HOT/WARM/COLD/NOT FIT.
3. Презентует продукт через выгоды (не фичи), подбирает тариф, даёт социальное доказательство.
4. Обрабатывает возражения по фреймворку ARPA (Acknowledge → Reframe → Prove → Ask).
5. Доводит до целевого действия: оплата, заявка, звонок, демо.
6. Передаёт менеджеру сложных/VIP-клиентов с полным контекстом (хэндофф).
7. Владелец перехватывает любой диалог (/stop) и возвращает управление (/resume).

- `DEPLOY_PROMPT.md` — промпт для Claude Code.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — дальше он сам проведёт онбординг.

## Требуемые интеграции

Telegram (обязательно — бот или личный аккаунт). Рекомендуются: Gmail, Google Sheets / Notion (CRM), Google Calendar (запись на звонки).

## Ограничения

- Агент не принимает оплату — только ведёт к ссылке на оплату.
- Не звонит по телефону, не управляет браузером.
- Не инициирует диалоги (только отвечает на входящие в режиме бота).
- В режиме личного аккаунта — может писать первым, но только по команде владельца.
