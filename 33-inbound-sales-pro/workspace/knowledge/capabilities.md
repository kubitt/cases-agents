# Возможности агента Sales Pro

> Ваш справочный файл. Агент читает его перед работой и дополняет по ходу. Правьте свободно.

## Что агент умеет

### Продажи
- Отвечать клиентам в Telegram 24/7 (бот или личный аккаунт)
- Квалифицировать лидов по 4 критериям (Fit, Need, Budget, Urgency)
- Презентовать продукт через выгоды
- Обрабатывать возражения по фреймворку ARPA
- Доводить до целевого действия
- Передавать менеджеру с контекстом

### Работа с информацией
- Веб-поиск и чтение веб-страниц
- Работа с файлами
- Долговременная память между сессиями

### Автоматизации
- Задачи по расписанию (утренний отчёт, аналитика)
- Триггеры по событиям (новое письмо → квалификация)

## Доступные интеграции (OAuth, 1 клик)

- Gmail: Читать входящие, писать, отправлять, сортировать
- Google Календарь: Встречи, создание событий
- Google Диск: Файлы из облака
- Google Docs / Sheets: Документы и таблицы
- Slack: Каналы и сообщения
- Notion: Страницы и базы данных
- GitHub / GitLab: Задачи, ветки, PR
- Supabase: Базы данных
- Google Search Console, Meet, Slides: Аналитика, встречи, презентации

### Для продажника полезнее всего:
- **Gmail** — заявки из почты, follow-up
- **Google Sheets / Notion** — CRM
- **Google Календарь** — запись на звонки
- **Slack** — уведомления команде

Подключение: «Подключи Gmail» в чате или Мозг → Подключения → сервис → + Добавить.

## Чего НЕ умеет
- Звонить по телефону
- Управлять браузером
- Принимать оплату напрямую
- Постить в Instagram
- Отправлять WhatsApp
