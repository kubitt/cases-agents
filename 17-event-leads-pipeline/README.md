# Бейджи с выставки → пайплайн / Event Leads Pipeline

CSV со сканами бейджей после выставки → каждый контакт обогащён ресёрчем, отскорен и получает персональный follow-up в тот же день. Пока конкуренты неделю «разбирают визитки», вы уже в переписке.

## Ценность и ROI

Участие в выставке стоит $2000–10000; невыжатые контакты — потерянные деньги. Ручная обработка 200 сканов (ресёрч + письма) — 30+ часов. Агент делает это за одну сессию, горячим пишет в день выставки — когда вас ещё помнят. Окупаемость $29: 30–100x за один сезон мероприятий.

## Что делает

1. Принимает CSV со сканами бейджей (или фото-список визиток текстом) — через Drive-папку «Выставки» или файлом в чат.
2. Нормализует и дедуплицирует: имена, компании, должности, email.
3. Обогащает каждый контакт веб-ресёрчем: чем занимается компания, размер, свежие новости, зачем могли подойти к стенду.
4. Скорит A/B/C по вашему ICP: горячие / тёплые / нерелевантные.
5. Follow-up: горячим — персональное письмо в день выставки (со ссылкой на контекст разговора, если вы его пометили в CSV), тёплым — цепочка из 3 касаний. Всё черновиками в Gmail на утверждение.
6. Ведёт таблицу-пайплайн: статусы, ответы, следующие шаги; после мероприятия — отчёт по конверсии.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN.
- `DEPLOY_PROMPT.md` — промпт для Claude Code.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка, триггер Drive и динамические задачи.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — онбординг: ICP-критерии, шаблон письма, тест на 10 строках.

## Требуемые интеграции

Gmail, Google Sheets, Google Drive (обязательно). Канал: Telegram (рекомендуется).

## Ограничения (сознательные)

- Без интеграции с CRM и сендерами — выход в Sheets + черновики Gmail (перенос в CRM — экспортом; или кастомный MCP).
- Обогащение — по открытому вебу; личные профили соцсетей не скрейпятся.
- Сезонный агент: постоянных cron нет — активируется загрузкой файла, цепочки ведутся динамическими задачами на 5–12 дней.
