# Интеграции: Финансовый пакет + runway (аутсорс-CFO)

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Drive | Обязательно | Папка «Выгрузки» (триггер приёма файлов), архив отчётов «Финансы/Отчёты» | Чтение и запись |
| Google Sheets | Обязательно | Рабочая книга «Финансы — операции»: транзакции, остатки, P&L, runway-модель | Чтение и запись |
| Google Docs | Рекомендуется | Нарративная часть пакета: комментарии, одностраничник основателя | Чтение и запись |
| Telegram (канал) | Рекомендуется | Понедельничный runway, алерты о разрывах, готовность месячного пакета | Канал доставки |

Порядок подключения при онбординге: Drive → Sheets → Docs → Telegram.

Без Docs: нарратив и одностраничник — текстом в чат/канал, таблицы в Sheets. Без Telegram: всё в чат ASCN. Без триггера Drive: выгрузки файлом в чат или команда «проверь папку Выгрузки».

Чего сознательно нет: банковских API, QuickBooks, 1С, Stripe (нет в 13 коннекторах ASCN) — замена: регулярные выгрузки CSV/XLSX. Продвинутый вариант: кастомный MCP к вашей учётной системе. Факт остатка на счёте агент узнаёт только из выписки или вашего сообщения.
