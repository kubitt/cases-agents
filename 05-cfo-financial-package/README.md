# Финансовый пакет + runway (аутсорс-CFO) / CFO Financial Package & Runway

Из сырых выгрузок банка и CRM — месячный финансовый пакет с комментариями к отклонениям и живой 13-недельный прогноз кэша: вы всегда знаете, сколько недель жизни у бизнеса.

## Ценность и ROI

Part-time CFO или финконсультант — $500–3000/мес. Агент закрывает базу этой работы: P&L, движение денег, комментарии к отклонениям, одностраничник для основателя, runway с тремя сценариями. Знание «касса уходит в минус на неделе N» до того, как это случилось, спасает бизнес — это дороже любой подписки. Окупаемость $29/мес — 17–100x.

## Что делает

1. Принимает выгрузки (банк CSV/XLSX, CRM, таблицы) в Drive-папку «Выгрузки» (триггер): валидирует период, колонки и дубли, классифицирует операции по вашему плану счетов (workspace/knowledge/CHART_OF_ACCOUNTS.md).
2. Незнакомые операции не угадывает — спрашивает и превращает ваши ответы в правила классификации.
3. 3-го числа собирает пакет за прошлый месяц: P&L по категориям, движение денег, сверка остатков, комментарии к отклонениям (против прошлого месяца и среднего за 3 месяца).
4. Одностраничник для основателя: 5 цифр, «главное» месяца, до 3 комментариев, 1 решение на стол.
5. Каждый понедельник в 09:00 обновляет 13-недельный прогноз кэша: 3 сценария (база/оптимист/пессимист), Week-0 = фактический остаток на счёте.
6. Алерты первой строкой: runway < 12 недель, кассовый разрыв на горизонте, расхождение расчётного и фактического остатка.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN (IDENTITY, SOUL, ONBOARDING, скиллы, план счетов, шаблон отчёта).
- `DEPLOY_PROMPT.md` — промпт для Claude Code, который переносит всё в ASCN.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка по интеграциям и готовые тексты триггера и cron-задач.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — дальше он сам проведёт онбординг (интеграции, план счетов по вашей реальной выписке, первый runway).

## Требуемые интеграции

Google Drive (обязательно, триггер), Google Sheets (обязательно), Google Docs (рекомендуется). Канал: Telegram (рекомендуется).

## Ограничения (сознательные)

- Без прямых интеграций с банками, QuickBooks и 1С (нет в коннекторах ASCN) — вход только выгрузками CSV/XLSX в Drive или Sheets; продвинутый вариант — кастомный MCP.
- Не финансовый советник: не рекомендует инвестиции, кредиты и налоговые схемы — считает и показывает сценарии.
- Управленческие цифры для решений, не бухгалтерская отчётность для налоговой. Полный баланс не строит — только то, что видно из выгрузок (кассовый метод).
- Расчёты только из данных пользователя: пропуски и нестыковки показываются явно, а не «сглаживаются».
