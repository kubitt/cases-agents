# Market scan

> A skill = one repeatable job, written as a recipe. You can edit steps, limits, and formats in plain words.

Goal: a short, sourced read of the competitive field that ends in one clear market gap and 2–3 things the user can act on.

## Algorithm
1. Read knowledge/BUSINESS_PROFILE.md for the offer, audience, and named competitors. Ask for 2–5 competitors if none are on file.
2. Sort competitors into three tiers: direct (same offer, same audience), indirect (different offer, same need), secondary (adjacent or aspirational). Cover at least one of each when possible.
3. For each competitor, read only public pages via web search and reading — their site, public pricing, public reviews, news, and any openly viewable ad library page. Keep a source URL for every fact. Mark anything you cannot confirm as "unverified".
4. Capture per competitor: positioning line, 2 strengths, 2 weaknesses, price band, and their main offer or hook. No copying their words — analysis only.
5. Name the market gap the user can own, then 2–3 recommended actions. Save it to a Google Doc "Market Scan — <topic>" with a sources list at the end.

## Output
```
Market Scan — specialty coffee subscriptions (2026-07)
Direct: Bean Box — positioning "discovery, many roasters"; strengths: variety, gifting; weaknesses: price ($24+), no single-origin focus; source: ...
Indirect: local roastery cafes — strength: freshness + community; weakness: no shipping; source: ...
Secondary: grocery specialty brands — strength: price/availability; weakness: staleness; source: ...
Gap: fresh single-origin at a mid price with flexible pausing — under-served between premium discovery boxes and grocery.
Actions: 1) lead with roast date on the page; 2) add a $18 single-origin tier; 3) test a "pause anytime" message. (unverified: Bean Box churn)
```

## Edge cases
- A competitor page is unreachable or thin → note it, use what's public, and cap effort at a few minutes per competitor.
- Only paid tools would answer a question (traffic, exact ad spend) → say so and mark it a limitation; suggest the optional data API rather than guessing.
- Conflicting figures across sources → show the range and cite both, don't average into a false single number.
- No clear gap emerges → say so honestly and give the closest positioning opportunity instead of inventing one.
