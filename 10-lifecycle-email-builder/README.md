# Lifecycle-email конструктор / Lifecycle Email Builder (опубликовано в marketplace)

> 📢 Опубликован в ASCN Marketplace: [RU](https://ascn.ai/ru/templates/lifecycle-email-marketing-automation-agent) · [EN](https://ascn.ai/en/templates/lifecycle-email-marketing-automation-agent)

Полная система удержания клиентов из одного описания бизнеса: сегменты, ветки писем, копирайт, тайминги — пакетом, готовым к загрузке в вашу рассылочную платформу.

## Ценность и ROI

Email-маркетолог на фрилансе берёт $500–1500 за одну кампанию, агентство за систему lifecycle-цепочек — $2000+. Агент собирает welcome-, activation-, winback- и VIP-ветки за одну сессию и обновляет их по метрикам. Окупаемость $29: 17–50x — плюс сами цепочки возвращают клиентов и повторные продажи.

## Что делает

1. Из описания бизнеса строит карту lifecycle-сегментов: welcome, activation, winback, VIP — с условиями входа/выхода из веток.
2. Пишет полный копирайт всех писем каждой ветки в тоне бренда: темы (с A/B-вариантами), тело, CTA, тайминг отправки.
3. Собирает upload-ready пакет: Google Docs со всеми письмами + Sheets с настройками веток (сегмент, условие, задержка, письмо) — для переноса в любую ESP.
4. Для баз до ~200 контактов может отправлять напрямую через Gmail (с разносом по времени, после утверждения).
5. Раз в неделю разбирает метрики кампаний (открытия/клики — из выгрузки в Sheets) и предлагает правки слабых писем.
6. По команде добавляет новые ветки: брошенная корзина, реактивация, NPS-опрос.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN.
- `DEPLOY_PROMPT.md` — промпт для Claude Code.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка и тексты cron-задач.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — онбординг проведёт сам (бизнес, тон, сегменты, первая ветка сегодня).

## Требуемые интеграции

Google Docs, Google Sheets (обязательно), Gmail (для малых баз и черновиков). Канал: Telegram (рекомендуется).

## Ограничения (сознательные)

- Прямых интеграций с ESP (Mailchimp, Unisender и т.п.) нет — выход в формате upload-ready пакета; загрузка в ESP на стороне пользователя (или кастомный MCP к API вашей ESP).
- Метрики кампаний агент читает из выгрузок ESP в Sheets — автоматического сбора статистики нет.
- Массовые отправки через Gmail ограничены: >200 контактов — только через ESP (риск блокировки домена; агент предупреждает).
