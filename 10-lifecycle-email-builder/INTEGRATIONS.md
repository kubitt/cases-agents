# Интеграции: Lifecycle-email конструктор

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Docs | Обязательно | Тексты всех писем кампаний (один док = одна ветка) | Чтение и запись |
| Google Sheets | Обязательно | Карта сегментов, настройки веток, база контактов (для Gmail-режима), метрики из выгрузок ESP | Чтение и запись |
| Gmail | Рекомендуется | Отправка малым базам (<200) после утверждения; черновики тестовых писем | Черновики, отправка |
| Telegram (канал) | Рекомендуется | Утверждение писем, недельные разборы метрик | Канал доставки |

Порядок подключения при онбординге: Docs → Sheets → Gmail (опционально) → Telegram.

Без Gmail: только пакет для ESP, без прямой отправки. Без Telegram: всё в чат ASCN.

Чего сознательно нет: коннекторов ESP (Mailchimp/Klaviyo/Unisender — нет в 13 OAuth ASCN). Замена: upload-ready пакет + выгрузки метрик CSV в Sheets. Продвинутый вариант — кастомный MCP к API вашей ESP.
