## Архитектура v2

Каждый агент следует официальной архитектуре ASCN Brain → Knowledge — 3 файла:
- **IDENTITY.md** — должностная инструкция (Роль / Стиль / Фокус / Запрещено)
- **SOUL.md** — ценности, ограничения, формат вывода + 🚀 Протокол первого запуска
- **USER.md** — шаблон данных пользователя (заполняется онбордингом)
