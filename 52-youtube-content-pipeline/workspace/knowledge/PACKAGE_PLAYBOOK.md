## Scoring rubric (idea 1–10)
(plain "field: value" lines — no tables, the ASCN UI breaks them)
- Fit: how well it matches the channel and audience (weight this most): [FILL DURING ONBOARDING]
- Trend heat: how timely / rising the topic is right now: [FILL DURING ONBOARDING]
- Differentiation: how distinct our angle is vs what's already out there: [FILL DURING ONBOARDING]
- How to combine: simple average of the three, rounded; anything below fit 4 is not pitched.

## Thumbnail brief standards
- One clear focal idea, 2–4 word overlay, high contrast; always give 2 variations to test.

## Worked example
- Idea: "Cline vs Cursor for non-coders" — Fit 9, Trend heat 8, Differentiation 8 → score 8.
- Titles: "I Let 2 AI Coders Build the Same App (No Code Skills)" / "Cursor vs Cline: Which One Actually Ships?" / "The AI Coding Showdown for Non-Coders".
- Description: "Two AI coding tools, one task, zero coding. Here's which one shipped. Timestamps below. Tools linked."
- Tags: ai coding, cursor, cline, no code, vibe coding, ai tools 2026, build an app, non-technical founder.
- Thumbnail: split screen, overlay "WHICH WINS?"; variation B: shocked face + "$0 CODE".

## Rules
- Every fact or statistic in a package needs a source URL, or it is marked "unverified".
- The title must match the video; no misleading clickbait.
- No copying reference channels — inspiration and analysis only.
