# SaaS-аудитор расходов / SaaS Spend Auditor (опубликовано в marketplace)

> 📢 Опубликован в ASCN Marketplace: [RU](https://ascn.ai/ru/templates/ai-saas-spend-auditor-cut-subscription-costs) · [EN](https://ascn.ai/en/templates/ai-saas-spend-auditor-cut-subscription-costs)

Находит все подписки компании по квитанциям в почте и банковским выпискам, выносит вердикт по каждой и ловит продления заранее — лишние списания прекращаются.

## Ценность и ROI

Типовая находка первого аудита — $100–500/мес зомби-подписок, дублей и раздутых тарифов: забытый триал, два тула для одной задачи, 10 оплаченных мест при 3 активных. Подписка ASCN $29/мес окупается первым же запуском: экономия $100/мес = $1200/год против $348/год подписки, то есть 30–100x. Плюс исчезают «автосписания за год, которые заметили постфактум».

## Что делает

1. Разбирает Gmail за 12 месяцев: квитанции, инвойсы, письма о продлении — извлекает вендора, сумму, периодичность, дату продления.
2. Парсит банковские выписки CSV: находит регулярные списания, сверяет с почтой, ловит подписки без квитанций.
3. Ведёт таблицу аудита в Google Sheets: сервис, категория, цена, годовая стоимость, дата продления, вердикт.
4. Помечает дубли (два сервиса одной категории) и зомби (списания идут, использования нет).
5. По каждой подписке — вердикт keep / cut / renegotiate с обоснованием в одну строку и суммой экономии $/год.
6. Готовит письма на отмену и переторговку (RU/EN) — отправка только после вашего утверждения.
7. Ежедневно в 09:00 проверяет даты продления: алерт за 14 и за 3 дня до списания.
8. 1-го числа в 10:00 — месячный срез: сколько тратим, что нового появилось, сколько уже сэкономили.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN (IDENTITY, SOUL, ONBOARDING, скиллы, схема аудита и шаблоны писем).
- `DEPLOY_PROMPT.md` — промпт для Claude Code, который переносит всё в ASCN.
- `INTEGRATIONS.md` — справка по интеграциям. Тексты cron-задач — в workspace/AUTOMATIONS.md.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — дальше он сам проведёт онбординг и в конце сделает первый полный аудит с суммой экономии.

## Требуемые интеграции

Gmail (обязательно), Google Sheets (обязательно). Канал: Telegram (рекомендуется).

## Ограничения (сознательные)

- Нет прямого доступа к банку и биллингам вендоров — источники: квитанции в Gmail + выписки CSV, которые вы выгружаете сами. Для полного аудита этого достаточно.
- Агент не отменяет подписки сам: нет браузерной автоматизации (ограничение платформы). Он готовит письмо или пошаговую инструкцию отмены — финальный клик за вами.
- Оценка использования сервиса — по косвенным признакам и вашим ответам, не по внутренней аналитике вендора.
