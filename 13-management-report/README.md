Каждый понедельник — нарративный KPI-отчёт с комментариями к отклонениям и действиями, плюс интерактивный HTML-дашборд одним файлом: открывается в браузере без всяких BI-систем.

## Ценность и ROI

Аналитик на полставки — $500–1500/мес, BI-подписки — $100–500/мес, и всё равно кто-то тратит 3–5 часов в неделю на ручную сборку цифр в отчёт. Агент собирает отчёт и дашборд по расписанию из ваших же источников. При подписке $29/мес окупаемость 15–40x.

## Что делает

1. Понедельник 08:30 — недельный отчёт: KPI-таблица (факт / план / дельта WoW), комментарий к каждому отклонению больше 20%, ровно одно рекомендованное действие на кластер метрик (деньги / маркетинг / продукт / разработка).
2. 1-го числа в 09:00 — месячный отчёт: MoM-динамика, тренды за 6 месяцев, итоги месяца.
3. Источники: Google Sheets (любые ваши метрики), Search Console (поисковый трафик), GitHub (велосити разработки). Карта «метрика → где лежит» настраивается при онбординге.
4. Генерирует автономный HTML-дашборд: один файл в Drive, без внешних зависимостей и CDN, данные встроены, вкладки по кластерам, SVG-графики. Открывается локально в любом браузере — можно показать на планёрке без интернета.
5. Ведёт историю значений всех метрик в служебной таблице — из неё строятся тренды и спарклайны.
6. Доставка в Slack или Telegram; по запросу в любой момент — экспресс-срез «как дела по метрикам».

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN (IDENTITY, SOUL, ONBOARDING, скиллы, шаблоны).
- `DEPLOY_PROMPT.md` — промпт для Claude Code, который переносит всё в ASCN.
- `INTEGRATIONS.md` — справка по интеграциям. Тексты cron-задач — в workspace/AUTOMATIONS.md.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — дальше он сам проведёт онбординг (интеграции, карта метрик, цели, тестовый отчёт и первый дашборд).

## Требуемые интеграции

Google Sheets (обязательно), Google Drive (обязательно). Search Console и GitHub — опционально. Канал: Slack или Telegram (рекомендуется).

## Ограничения (сознательные)

- Stripe/Mixpanel/CRM/рекламные кабинеты нативно не подключаются — их цифры попадают в отчёт через выгрузки в Sheets (или кастомный MCP как расширенная настройка).
- Дашборд — офлайн-файл, а не живая веб-страница: обновляется по расписанию перезаписью файла в Drive.
- Агент не выдумывает пропуски в данных: недоступный источник в отчёте помечается «n/d» с причиной.
