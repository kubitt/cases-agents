# HTML-дашборд

Цель: один файл dashboard.html, который открывается двойным кликом в любом браузере — без интернета, серверов и BI.

## Жёсткие требования к файлу
- Автономность: весь CSS и JS — инлайн в файле. Никаких CDN, внешних шрифтов, картинок по URL, счётчиков. Самопроверка перед сохранением: в файле нет ни одного http(s)-URL в src/href.
- Данные — JSON-объект внутри тега script: история метрик из «KPI-история» за 12 последних периодов.
- Графики — инлайн SVG, разметку генерируй сам (полилиния тренда + точки); chart-библиотеки запрещены.
- Размер ≤ 1 МБ. Только агрегированные значения — никаких сырых строк с именами клиентов или сотрудников.

## Структура дашборда
1. Шапка: название компании, период данных, дата генерации.
2. Вкладки по кластерам (деньги / маркетинг / продукт / разработка) — переключение на чистом JS, активная вкладка запоминается в localStorage.
3. На вкладке: карточки KPI — текущее значение, дельта к прошлому периоду (зелёная/красная с учётом направления метрики из METRICS_MAP.md: рост churn — красный, хоть и рост), под значением SVG-спарклайн 12 периодов, у метрик с целью — линия цели.
4. Ниже — таблица истории метрик кластера за 12 периодов.
5. Блок «Действия недели» — из последнего kpi-report.
6. Подвал: «Сгенерировано агентом ASCN, данные на <дата>» + список метрик со статусом n/d.

## Алгоритм
1. Прочитай «KPI-история» (12 последних периодов на метрику) и knowledge/METRICS_MAP.md (кластеры, направления, цели, приоритеты).
2. Собери HTML по структуре выше. Прогони самопроверку автономности и размера.
3. Сохрани в Drive-папку «Дашборды» как dashboard.html — перезапиши существующий (файл всегда актуальный срез; история живёт в Sheets). При месячном отчёте дополнительно сохрани копию dashboard-<YYYY-MM>.html.
4. Пришли ссылку на файл + одну строку, что изменилось («добавлен спарклайн churn, лиды ушли в красное»).

## Краевые случаи
- Истории мало (первые 2–3 недели) → рисуй что есть, подпиши «данные накапливаются».
- Метрик больше 30 → карточками только priority=1 из METRICS_MAP.md, остальные таблицей в конце вкладки.
- Метрика n/d → серая карточка с подписью «нет данных», не рисуй ноль: ноль и «нет данных» — разные вещи.
- Drive недоступен → не теряй работу: пришли файл прямо в чат вложением и алерт про переподключение.
