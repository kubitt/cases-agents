# Интеграции: Фабрика локальных SEO-страниц

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Sheets | Обязательно | Матрица «услуга x район»: приоритеты, статусы, URL, даты публикации | Чтение и запись |
| Google Docs | Обязательно | Тексты страниц в читаемом виде для правок и утверждения | Чтение и запись |
| Google Drive | Обязательно | HTML-файлы пакета, папка проекта, инструкция по публикации | Чтение и запись |
| Google Search Console | Рекомендуется | Приоритизация по реальным запросам, контроль индексации опубликованного | Чтение |
| Telegram (канал) | Опционально | Уведомления о готовности батча | Канал доставки |

Встроенный веб-поиск и чтение страниц — сбор локальных деталей районов и оценка конкуренции, OAuth не требуют.

Порядок подключения при онбординге: Sheets → Docs → Drive → Search Console → Telegram.

Без Search Console: приоритеты — только по косвенным сигналам веб-поиска; индексацию проверяет пользователь. Без Telegram: уведомления в чат ASCN.

Чего сознательно нет: публикации в CMS (WordPress/Tilda/Bitrix) — нет таких коннекторов и нет браузера. Заменено: пакет Docs + HTML в Drive + инструкция; продвинутый вариант — кастомный MCP к REST API вашей CMS (например WordPress) — тогда команду «опубликуй батч» можно отдать агенту.
