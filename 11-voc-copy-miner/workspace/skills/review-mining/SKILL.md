# Майнинг отзывов

Цель: банк дословных формулировок клиентов с источниками. Дословность — главная ценность: не улучшать, не сокращать, не исправлять грамматику.

## Алгоритм
1. Источники — knowledge/MINING_SOURCES.md. Для каждого: открой страницы отзывов продукта (свои и конкурентов), читай от новых к старым.
2. Из каждого содержательного отзыва извлекай фразы-кандидаты: боль («задолбались ждать менеджера по три часа»), желание («хочу чтобы просто работало и не звонили»), возражение («думал дорого, оказалось…»), восторг («спасли наш переезд»). Смайлы и междометия сохраняются.
3. Фильтр: фраза конкретная (есть ситуация/деталь), не заказная (шаблонные «всё супер, рекомендую» — мимо), 3–25 слов.
4. Запись в Sheets «VoC-банк»: quote (дословно), type, theme (нормализуй: доставка / цена / поддержка / качество / …), product (наш / конкурент N), source, url, date отзыва.
5. Дедуп: почти одинаковые фразы с одной площадки — одна запись с пометкой частоты (freq).
6. Итог прогона: сводка по темам (каких болей больше), топ-5 фраз, аномалии (новая тема, всплеск негатива у конкурента).

## Нормы объёма
Мини-банк (онбординг): 20 цитат. Полный: 100–300. Больше 300 — по согласованию, отдача падает.

## Краевые случаи
- Площадка не открывается / антибот → пометь в MINING_SOURCES.md (blocked, дата), не мучай повторными попытками, сообщи пользователю.
- Отзывов о продукте пользователя мало → усиль конкурентов и общие обсуждения ниши (форумы, Q&A) — боли ниши тоже валидны, пометь product=ниша.
- Не-русскоязычные отзывы → сохраняй в оригинале + перевод в соседней колонке.
