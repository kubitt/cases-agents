# Фреймворки сборки копирайта

## PAS (холодный трафик, объявления)
Problem: формула боли из топ-темы — дословной лексикой клиентов.
Agitate: последствие из цитат («в итоге потеряли клиента / переплатили дважды»).
Solve: механизм продукта + подтверждённое обещание. CTA-глагол.
Пример сборки: цитата «задолбались ждать менеджера по три часа» → заголовок «Менеджер отвечает за 3 минуты, а не за 3 часа».

## JTBD-блоки (лендинг)
Порядок экранов: 1) первый экран = формула мечты топ-темы + продукт как способ; 2) «знакомо?» = 3 боли цитатами; 3) механизм (как решаем, без жаргона); 4) доказательства (реальные отзывы — полные, со ссылками); 5) возражения из банка → честные ответы; 6) CTA.

## Углы отстройки
Из болей в отзывах конкурентов, отсутствующих у нас: «Часто жалуются на X. У нас: <как решено>». Без имён конкурентов в публичном тексте.

## Гигиена
Один блок = одна тема банка. Не смешивать боли разных сегментов в одном экране. Слова-паразиты копирайтеров («уникальный», «инновационный») — запрещены: их нет в языке клиентов.
