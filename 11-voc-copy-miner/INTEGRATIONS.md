# Интеграции: VoC-майнер

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Sheets | Обязательно | Банк цитат: фраза, тема, тип (боль/желание/возражение), источник, ссылка, дата | Чтение и запись |
| Google Docs | Обязательно | Готовый копирайт: карта хуков, блоки лендинга, объявления | Чтение и запись |
| Telegram (канал) | Рекомендуется | Результаты майнинга, копирайт на утверждение | Канал доставки |

Встроенное (без настройки): веб-поиск, чтение веб-страниц — основной инструмент сбора отзывов.

Порядок подключения при онбординге: Sheets → Docs → Telegram.

Без Telegram: всё в чат ASCN. Чего сознательно нет: API отзовиков и парсеров (не требуются — чтение открытых страниц); браузерной автоматизации нет, поэтому площадки за логином недоступны.
