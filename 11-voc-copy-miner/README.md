# Голос клиента → продающий копирайт (VoC-майнер) / VoC Copy Miner

Собирает дословные фразы клиентов из отзывов — ваших и конкурентов — и превращает их в хуки, заголовки и копирайт, который звучит языком покупателя, а не «AI-прозой».

## Ценность и ROI

Копирайтер + кастдев-исследование стоят $500–2000 за проект. Агент делает банк из сотен реальных цитат с источниками и готовый копирайт лендинга/объявлений за одну сессию. Окупаемость $29: 17–70x. Бонус: копирайт на языке клиентов конвертирует лучше — это подтверждённая практика (VoC-подход).

## Что делает

1. Мониторит открытые площадки с отзывами по вашим продуктам и конкурентам (список настраивается при онбординге).
2. Извлекает дословные фразы боли, желания, возражения, восторга — в банк цитат в Google Sheets с источником и ссылкой.
3. Группирует цитаты по темам, строит карту хуков: какие формулировки повторяются чаще всего.
4. Пишет копирайт из банка: заголовки, блоки лендинга, тексты объявлений — по фреймворкам PAS/JTBD, дословным языком клиентов.
5. Отдельно собирает возражения из отзывов конкурентов — готовые углы отстройки («в отличие от X, у нас…»).
6. Раз в месяц пополняет банк новыми отзывами (опционально).

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN.
- `DEPLOY_PROMPT.md` — промпт для Claude Code.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка и тексты cron-задач.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — онбординг сам проведёт (продукт, площадки, тестовый мини-банк из 20 цитат сегодня).

## Требуемые интеграции

Google Sheets, Google Docs (обязательно). Веб-поиск и чтение страниц — встроенные. Канал: Telegram (рекомендуется).

## Ограничения (сознательные)

- Только открытые площадки: закрытые группы и площадки с жёсткой антибот-защитой (проверяется на онбординге) недоступны — агент честно помечает, что не смог прочитать.
- Агент не публикует контент сам и не пишет фейковые отзывы (запрещено в SOUL).
- Тексты требуют финальной вычитки владельцем: язык клиентов ≠ юридическая корректность обещаний.
