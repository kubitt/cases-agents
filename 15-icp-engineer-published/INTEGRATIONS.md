# Интеграции: ICP-инженер из win/loss данных

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Sheets | Обязательно | Таблица «Win-Loss сделки» — вход данных; агент сам создаёт шаблон с нужными колонками | Чтение и запись |
| Google Docs | Опционально | Копия живого ICP-документа — удобно шарить команде | Чтение и запись |
| Telegram (канал) | Опционально | Уведомления ежемесячной проверки пополнения | Канал доставки |

Встроенный веб-поиск используется для обогащения персон (боли ниши, каналы, триггеры) — отдельного подключения не требует.

Порядок подключения при онбординге: Sheets → (Docs по желанию).

Без Docs: ICP-документ живёт в knowledge/ICP.md воркспейса и выдаётся в чат по команде. Без Sheets: можно прислать CSV прямо в чат, но ежемесячная проверка пополнения работать не будет (нет источника для сравнения).

Чего сознательно нет: CRM-коннекторов (amoCRM, Bitrix24, HubSpot, Pipedrive) — заменено экспортом в Sheets/CSV, инструкции в knowledge/DEALS_SCHEMA.md. CRM при желании подключается кастомным MCP.
