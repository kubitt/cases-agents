# ICP-инженер из win/loss данных / ICP Engineer

Выводит портрет идеального клиента из вашей реальной истории сделок: какие сигналы статистически отличают выигрыши от проигрышей — данные вместо мнения основателя.

## Ценность и ROI

Win/loss-ресёрч у консультантов стоит $2000–5000 и занимает недели. Агент делает первый разбор за одну сессию из выгрузки CRM в таблицу — и честно помечает, где данных мало. Уточнённый ICP поднимает конверсию всего аутбаунда и маркетинга (в связке с Outbound-машиной её скоринг становится точнее). Окупаемость подписки $29/мес — 70x+ на одном разовом проекте.

## Что делает

1. Принимает выгрузку сделок в Google Sheets или CSV (минимум 20–30 закрытых сделок; схема колонок — knowledge/DEALS_SCHEMA.md, есть инструкции экспорта из amoCRM/Bitrix24/HubSpot/Pipedrive).
2. Проверяет качество данных до анализа: пропуски, разнобой значений, дубли — и согласует правки, а не чинит молча.
3. Считает win rate по срезам: ниша, размер, гео, источник, роль ЛПР, размер чека, цикл сделки.
4. Честная статистика на малых выборках: сегменты с n<5 помечаются «мало данных», у каждого вывода — уровень уверенности.
5. Выявляет анти-ICP: где стабильно проигрываете и почему (частоты причин поражений, цитаты из notes).
6. Строит 1–3 скоринговые персоны для аутбаунда: боли, триггеры, возражения с ответами, где обитают, таблица скоринга на 100 баллов с порогами.
7. Ведёт живой ICP-документ (knowledge/ICP.md + копия в Google Docs): версии с датами, дельта «что изменилось» при каждом обновлении.
8. Опционально: 1-го числа в 11:00 проверяет пополнение таблицы и предлагает переанализ.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN (IDENTITY, SOUL, ONBOARDING, скиллы, схема сделок и шаблон ICP-документа).
- `DEPLOY_PROMPT.md` — промпт для Claude Code, который переносит всё в ASCN.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка по интеграциям и текст опциональной cron-задачи.

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — онбординг короткий (~5 минут + выгрузка сделок), полный анализ и персоны вы получите в первой же сессии.

## Требуемые интеграции

Google Sheets (обязательно), Google Docs (опционально). Встроенный веб-поиск — без отдельного подключения.

## Ограничения (сознательные)

- Без прямых CRM-интеграций (нет коннекторов) — вход через экспорт в Sheets/CSV; экспорт занимает 5 минут, агент подскажет как. CRM при желании подключается кастомным MCP.
- Статистика на 20–50 сделках — это направленные сигналы, а не «научная значимость»: агент маркирует уверенность каждого вывода и не выдаёт шум за инсайт.
- Персоны обогащаются веб-ресёрчем ниши; всё, что не из ваших данных, помечено [гипотеза] и подлежит проверке аутбаундом.
