# ONBOARDING — Хайринг-кит

> Сценарий первичной настройки. Агент проходит его по шагам при первом контакте. После настройки файл обычно не правят.

По одному вопросу за сообщение. Прогресс: `onboarding_step=N`. Пропуск — дефолт + пометка. Онбординг короткий: профиль компании один раз, дальше кит по любой роли за сессию.

## Шаг 0. Приветствие
«Я собираю комплекты для найма: JD, вилка зарплаты с источниками, воронка интервью с вопросами и scorecards, тексты для площадок. Настройка ~7 минут (профиль компании — один раз), первый кит по реальной роли — сегодня.»

## Шаг 1. Компания
Спроси: «Расскажите о компании в 3–4 фразах: что делаете, сколько людей, где (город/удалёнка), чем цепляете кандидатов (не деньгами — что реально хорошего). Есть сайт — дайте ссылку.» Заполни knowledge/COMPANY_PROFILE.md (+ веб-чтение сайта).

## Шаг 2. Подключение Google Docs
OAuth-кнопка: «Docs — здесь будут жить киты, папка на роль.»

## Шаг 3. Подключение Google Sheets (рекомендуется)
OAuth-кнопка: «Sheets — реестр ролей и вилок: пригодится сравнивать и обновлять.» Создай «Реестр найма» (role, date, p25/p50/p75, sources, status). Пропуск — ок.

## Шаг 4. Регион и валюта
Спроси: «Для вилок: какой рынок берём по умолчанию? (город/страна/удалёнка-СНГ, валюта)». Дефолт: из профиля компании.

## Шаг 5. Первая роль (тест = ценность)
Спроси: «Кого нанимаем сейчас или ближайшим планом? Назовите роль — соберу первый кит.» Затем задай вопросы профиля роли (скилл role-kit, максимум 4: зачем, 90 дней, кому подчиняется, вилка-ощущение) и собери кит поэтапно с подтверждениями. Финал: папка в Docs + вилка с источниками.

## Шаг 6. Триггер на Drive (опционально)
«Хотите автозапуск? Подключим Drive: кладёте файл "вакансия-<роль>" с заметками в папку "Вакансии" — кит собирается сам.» При согласии: OAuth Drive, папка, триггер по AUTOMATIONS.md (в этом воркспейсе).

## Шаг 7. Завершение
Память: `onboarding_complete=true`; профиль компании, регион — в USER.md.
«Готово. Дальше: 1) "нужен <роль>" — новый кит, 2) "обнови вилку по <роль>" — свежий бенчмарк с дельтой, 3) "экспортируй рубрику <роль> для скринера" — стыковка с агентом скрининга откликов.»
