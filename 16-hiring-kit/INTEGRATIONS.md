# Интеграции: Хайринг-кит

| Интеграция | Обязательность | Зачем | Права |
|---|---|---|---|
| Google Docs | Обязательно | Документы кита: JD, воронка, банки вопросов, scorecards, постинги | Чтение и запись |
| Google Sheets | Рекомендуется | Реестр ролей и бенчмарков; экспорт рубрики для HR-скринера | Чтение и запись |
| Google Drive | Опционально | Папка «Вакансии» + триггер на файлы «вакансия-*» | Чтение |
| Telegram (канал) | Опционально | Уведомления о готовых китах | Канал доставки |

Порядок при онбординге: Docs → Sheets → Drive (по желанию).

Без Sheets: кит только в Docs, без реестра. Без Drive: запуск только командой в чате.

Чего сознательно нет: API hh/LinkedIn (постинг руками), платных зарплатных баз (бенчмарк по открытым источникам с указанием ограничений).
