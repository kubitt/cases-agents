# Хайринг-кит под ключ / Hiring Kit Builder

Назовите роль — получите полный комплект для найма: JD, вилку зарплаты с источниками, 4-этапную воронку интервью с банками вопросов, scorecards и тексты для джоб-бордов. Всё согласовано между собой, за одну сессию.

## Ценность и ROI

HR-консультант берёт за такой комплект $500–1500; неправильная вилка стоит потерянного кандидата или переплаты $1000+/мес; несогласованные интервью — найма «по ощущениям». Окупаемость $29: 17–50x на первой же роли.

## Что делает

1. Собирает профиль роли из короткого диалога: зачем нанимаем, что человек должен сделать за первые 90 дней.
2. Пишет JD: результат-ориентированную (что сделает, а не «требования: стрессоустойчивость»), в тоне компании.
3. Бенчмарк зарплаты по открытым источникам: вилка p25/p50/p75 с цитированием источников и датой, честные пометки о разбросе данных.
4. Проектирует воронку интервью: 4 этапа (скрининг → профессиональное → кейс/тестовое → финал с ценностями), банк вопросов на каждый, критерии оценки.
5. Scorecards: карточки оценки, замаппленные на требования JD, — интервьюеры оценивают одно и то же.
6. Тексты постингов под каналы: hh, LinkedIn-стиль, телеграм-каналы вакансий — из одной JD.
7. Опционально: триггер на Drive — файл «вакансия-*» в папке → собрать кит автоматически.

## Состав папки

- `workspace/` — копируется в воркспейс агента ASCN.
- `DEPLOY_PROMPT.md` — промпт для Claude Code.
- `INTEGRATIONS.md`, `AUTOMATIONS.md` — справка (агент почти без cron: one-shot инструмент).

## Быстрый деплой

1. Откройте Claude Code в этой папке.
2. Вставьте содержимое `DEPLOY_PROMPT.md`.
3. Напишите агенту «Привет, начнём настройку» — онбординг: профиль компании один раз, дальше «нужен маркетолог» → кит.

## Требуемые интеграции

Google Docs (обязательно), Google Sheets, Google Drive (рекомендуются). Веб-поиск — встроенный.

## Ограничения (сознательные)

- Бенчмарк — по открытым источникам (агрегаторы вакансий, обзоры зарплат): точность уровня «рыночная вилка», не замена платным опросам зарплат; агент явно указывает источники и их свежесть.
- Постинг на джоб-борды — руками пользователя (нет API-коннекторов; тексты готовы к вставке).
- Скрининг откликов — не эта роль: для потока резюме есть отдельный агент HR-скринер PRO (шаблон 07), кит стыкуется с ним напрямую (рубрика экспортируется).
